## 🧠 What is FitGuard AI?

FitGuard AI is a multimodal fitness agent that combines real-time visual analysis, voice coaching with Gemini Live API, and Brain Endurance Training (BET) — a neuroscience technique to elevate brain frequency to 50Hz before working out.

**It solves 3 real problems:**
- 30–70% of recreational athletes suffer preventable injuries
- 70% abandon routines due to mental burnout
- Lack of accessible, personalized coaching

## ✨ Features

### ⚡ Pre-Workout
- Mental state and energy level analysis
- Interactive BET Protocol (memory, math, breathing)
- Brain frequency estimation (Hz)
- Personalized training plan generated by Gemini

### 🏋️ During Workout
- Real-time posture analysis with Gemini Vision
- Biomechanical error detection and injury risk alerts
- BET micro-tasks between sets to maintain 50Hz
- Real-time voice coach powered by Gemini Live API

### 🌙 Post-Workout
- Holistic recovery analysis
- Personalized nutrition and rest plan
- Next-day BET protocol recommendation

## 🧬 Brain Endurance Training (BET)

BET is a scientifically-validated technique that trains the brain under cognitive fatigue to improve mental resilience and athletic performance. FitGuard AI integrates BET at 3 key moments:

1. **Pre-workout:** Activation protocol to reach 50Hz brain frequency
2. **During workout:** Micro-tasks between sets to maintain optimal state
3. **Post-workout:** Recovery BET to prepare for the next session

The protocol includes 4 steps:
- **Working Memory** — Reverse a number sequence under time pressure
- **Mental Math** — Solve 3 arithmetic operations in 45 seconds
- **Pattern Recognition** — Identify the next number in a visual sequence
- **Neural Breathing** — 4-4-4 box breathing to synchronize brain waves
